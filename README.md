## 教程
1.下载index.html文件
2.宝塔用户将html文件放在pubilc文件夹下面
3.访问机场网址例如：xxx.com/index.html
## 优点
✅  简单部署
✅  外观好看
✅  自定义效果强
## 缺点
❌  就一个html文件
❌   使用是要加上/index.html
但是各位缺点我们会快马加鞭改进
